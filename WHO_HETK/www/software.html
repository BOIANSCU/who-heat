Software

The Health Equity Assessment Toolkit was developed using R: A Language and Environment for Statistical Computing and shiny: Web Application Framework for R. Additional R packages used to support the toolkit include:

  • countrycode: Convert Country Names and Country Codes.
  • digest: Create Cryptographic Hash Digests of R Objects.
  • ggplot2: elegant graphics for data analysis.
  • plyr: The Split-Apply-Combine Strategy for Data Analysis.
  • RSQLite: SQLite Interface for R.
  • shinyBS: Twitter Bootstrap Components for Shiny.
  • shinythemes: Themes for Shiny.

The standalone version of the toolkit uses the Portable Apps version of the Mozilla Firefox browser.
